# Gallstone Risk Inference

Reproducible analysis of gallstone risk factors using a clinical dataset (UCI Gallstone).

## Structure
- notebooks/ — Jupyter notebook (R kernel)
- data/ — input data (Excel/CSV)
- figs/ — figures saved from the notebook
- outputs/ — rendered HTML exports

## Reproduce
1) Launch Jupyter Lab: `jupyter lab`
2) Open `notebooks/gallstone_analysis.ipynb`
3) Ensure R has packages: tidyverse, readxl, broom
4) Data source: UCI ML Repository — Gallstone (accessed Oct 24, 2025)
